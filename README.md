# Proj_Conexao_Bolsa
Repositório do Projeto Conexão com a Bolsa de Valores. Focado em integrações via API e Simulações que maximizam o Rendimento.

O projeto tem duas frente: API/Conexão directa com os valores de cotas na Bolsa de Valores B3 e; Otimizador de rendimentos que utiliza técnicas de PLI (Programação Linear Inteira). 

Projeto iniciado em 16/03/2026 para testar e revisar os conhecimentos do autor.
